# The Platform Space

A practical knowledge base and AI assistant for Platform Product Managers — built on real experience across startup, scale-up, and public sector environments.

## What's in here

- **`pm-knowledge-bank.md`** — Frameworks, STAR stories, metrics, principles, and templates for platform PMs. Covers strategy, API products, GenAI governance, developer discovery interviews, and more.

## Part of The Platform Space

This repo is the foundation for [The Platform Space](https://theplatformspace.substack.com) — a newsletter and resource hub for platform PMs navigating developer experience, infrastructure strategy, and AI platform work.

## Contributing

Frameworks and templates are intended to be reusable and environment-agnostic. PRs and feedback welcome.
